# Noster Front

Front end React do Noster, um software de engenharia de decisao com IA.

## Scripts

- `npm run dev`: inicia o ambiente local.
- `npm run build`: valida TypeScript e gera build de producao.
- `npm run lint`: executa ESLint.
- `npm run format`: formata os arquivos com Prettier.

## Estrutura

- `src/app`: configuracoes da aplicacao, rotas e providers globais.
- `src/pages`: entradas das telas principais.
- `src/layouts`: layouts reutilizaveis, incluindo a base autenticada.
- `src/features`: dominios do produto, como reunioes, decisoes, insights e auth.
- `src/shared`: componentes, estilos, constantes e utilitarios compartilhados.
